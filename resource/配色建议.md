# 配色建议（Color Design Specification）

## 设计理念

本项目整体视觉定位为：

> **Apple × Nothing × Linear × Life is Strange**

核心关键词：

* 克制（Minimal）
* 忧郁（Melancholy）
* 冷静（Calm）
* 沉浸（Immersive）
* 电影感（Cinematic）
* 蓝色低饱和（Low Saturation Blue）

页面不追求高对比度，而是利用低饱和配色、柔和光影与雾化玻璃效果营造一种安静且富有情绪表达的体验。

整体视觉应让用户产生一种：

> 「坐在下雨天的车里，看着窗外森林缓缓掠过。」

而不是：

> 「这是一个普通的现代网页。」

---

# 配色原则

遵循以下原则：

* 避免纯白（#FFFFFF）
* 避免纯黑（#000000）
* 避免高饱和蓝色
* 避免高亮霓虹色
* 避免刺眼的文字对比

所有颜色都应略带蓝灰色调，与整体背景保持统一。

---

# 背景颜色（Background）

推荐使用深蓝灰渐变作为全站基础背景。

```css
background: linear-gradient(
    180deg,
    #324B66 0%,
    #2B425B 35%,
    #25384D 70%,
    #21344A 100%
);
```

特点：

* 冷色调
* 低饱和
* 不抢主体
* 与雨天森林背景自然融合

---

# 一级标题（H1）

推荐颜色：

```css
#E8EEF5
```

说明：

接近白色，但加入轻微蓝灰色调。

特点：

* 高级
* 柔和
* 不刺眼
* 保持 Apple 风格

示例：

```
The Rain Never Stops
```

---

# 二级标题（H2）

推荐颜色：

```css
#C9D5E3
```

相比 H1 略暗。

适合作为：

* 小标题
* Section 标题
* 卡片标题

---

# 正文文字（Body）

推荐颜色：

```css
#9FAFC2
```

或：

```css
rgba(159,175,194,0.85)
```

特点：

* 阅读舒适
* 不喧宾夺主
* 与背景自然融合

正文应像雾一样存在，而不是跳出页面。

---

# 辅助文字（Caption）

推荐颜色：

```css
#7B8798
```

或：

```css
rgba(123,135,152,0.75)
```

适用于：

* Footer
* 时间
* 标签
* 描述文字
* Secondary Info

例如：

```
© 2026 Mao's Digital World
```

---

# 强调色（Accent）

推荐颜色：

```css
#BFD8FF
```

用途：

* Hover
* Link
* Active
* Focus

不要使用高饱和蓝色。

Hover 建议仅产生轻微颜色变化。

例如：

```
Normal:
#E8EEF5

Hover:
#BFD8FF
```

---

# 按钮设计（Button）

推荐采用玻璃拟态设计。

背景：

```css
rgba(255,255,255,0.08)
```

边框：

```css
rgba(255,255,255,0.12)
```

文字：

```css
#E8EEF5
```

Hover：

```css
background:
rgba(255,255,255,0.12);
```

避免：

* 实心蓝按钮
* 高饱和按钮
* 发光按钮

---

# 雾玻璃（Fog Glass）

推荐参数：

背景：

```css
background:
rgba(255,255,255,0.06);
```

边框：

```css
border:
1px solid rgba(255,255,255,0.08);
```

模糊：

```css
backdrop-filter:
blur(20px);
```

阴影：

```css
box-shadow:
0 8px 32px rgba(0,0,0,.12);
```

原则：

不要 Glow。

不要发光。

不要霓虹。

保持安静且具有呼吸感。

# 最终视觉目标

整个网站的视觉体验应接近：

* Apple 官网的克制感
* Nothing 的留白与简洁
* Linear 的高级 UI
* 雨天车窗后的蓝色森林
* 带有电影镜头语言的沉浸式交互体验

用户打开页面时，应感受到：

> 「我仿佛坐在一辆停靠在森林边的车里，窗外下着雨，玻璃蒙着雾，整个世界安静而忧郁。」

而不是：

> 「这是一个普通的模板网站。」
