# 资料提交说明

这个文件夹用于收集你的真实文字信息和图片素材。当前项目里的姓名、经历、照片、音乐、食物、旅行等内容仍是占位内容，后续会按这里的资料替换到页面中。

## 建议填写顺序

1. 先填写 `01-文字信息/01-首页与开场.md`
2. 再填写 `01-文字信息/02-关于我.md`
3. 再填写 `01-文字信息/03-生活轨迹.md`
4. 然后补充影像、旅行、音乐、食物和页脚链接
5. 最后把图片放入 `02-图片素材` 下对应文件夹

## 图片格式建议

- 推荐格式：`.jpg` / `.jpeg` / `.png` / `.webp`
- 建议宽度：至少 1600px，横图优先
- 风格建议：低饱和、自然光、真实、有情绪，但不要过暗到看不清主体
- 上传前建议检查 EXIF 信息，避免公开精确定位、设备序列号等隐私信息

## 命名建议

尽量使用清晰的英文或拼音文件名，例如：

- `about-portrait.jpg`
- `gallery-01-rainy-night.jpg`
- `travel-02-dali-lake.jpg`
- `fog-window-clear.jpg`
- `fog-window-fogged.jpg`

## 重要说明

这里的资料文件不会自动替换页面。你把内容填好、图片放好后，我会根据这些资料去修改项目代码和图片引用。
